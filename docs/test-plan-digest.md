## Affected Pages/Routes
- `xx digest` CLI command — full pipeline from tweet pull to formatted output
- `xx digest --repo PATH` — work context extraction from a git repo
- `xx digest --since 24h --count 5` — time filtering and result truncation

## Key Interactions to Verify
- _parse_since() handles all formats: '24h', '3d', '1w', ISO date, invalid input
- context.py reads git log, README, deps from a real (or fixture) git repo
- digest.py filters tweets by created_at, sorts by score, truncates to --count
- format.py print_digest() renders adopt/avoid/copy tags, header stats, empty state
- Missing OPENAI_API_KEY produces a clear error before any API call
- Rate limit on digest shows a message mentioning shared quota with feed

## Edge Cases
- --repo path that isn't a git repo → warning + runs without context
- --repo path that doesn't exist → error
- Zero tweets returned from API → skip LLM call, print message
- All tweets score < 7 → empty digest, show stats
- README > 200 lines → truncated
- Fresh repo with no commits → empty git log handled
