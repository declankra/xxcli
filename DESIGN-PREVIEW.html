Implementation Notes

How this translates to Rich + Python

xx — implementation
Rich Theme (centralized styles)
# src/xxcli/theme.py
from rich.theme import Theme

xx_theme = Theme({
    "xx.author":  "bold",
    "xx.handle":  "dim",
    "xx.content": "",
    "xx.metrics": "dim",
    "xx.accent":  "cyan",
    "xx.success": "green",
    "xx.error":   "red bold",
    "xx.warning": "yellow",
    "xx.key":     "cyan",
    "xx.dim":     "dim",
})

console = Console(theme=xx_theme)
Pipe-Friendly Output
# Respect NO_COLOR and TTY detection
# Rich handles this automatically:
# - Detects TTY, strips markup when piped
# - Respects NO_COLOR env var
# - force_terminal=False by default

console = Console(
    theme=xx_theme,
    highlight=False,  # no auto-highlighting
)
TUI Framework Direction
Current: Rich (print-style output)
Future:  Textual (full TUI, same ecosystem)

Textual is built on Rich by the same team.
Same theme tokens, same color system.
Migration path: Rich panels → Textual widgets.
The design system works for both.
